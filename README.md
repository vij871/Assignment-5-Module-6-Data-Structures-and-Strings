# Assignment 5 – Module 6: Data Structures and Strings in Python

## Task 1: Student Marks Dictionary
This program:
- Creates a dictionary with student names as keys and marks as values
- Prompts the user to enter a student name
- Displays the student’s marks if found
- Shows an appropriate message if the student does not exist

File: `task1_student_marks.py`

## Task 2: List Slicing Demonstration
This program:
- Creates a list of numbers from 1 to 10
- Extracts the first five elements
- Reverses the extracted list
- Prints both the extracted and reversed lists

File: `task2_list_slicing.py`

## How to Run
Ensure Python 3 is installed, then run:

```bash
python task1_student_marks.py
